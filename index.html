BMWS!

Brain Machine Wellness Solutions, nace como una solución a los problemas de rendimiento académico que tienen los estudiantes. El producto que pretendemos vender es un dispositivo inteligente que correponde a un cintillo EEG, que funciona en conjunto con una app movil.

Nuestra Solucion.

EL EEG (electroencefalógrafo) se encarga de monitorear la actividad cerebral y enviar la informacíon en tiempo real a la app, la cual se encarga de procesar los datos y generar el estímulo correspondiente.

¿Como funciona?

Primero, cuando el usuario vaya a estudiar, iniciará la app y la dejerá andando mientras tanto. Ésta se encargará de reproducir un sonido abrupto de manera periódica cada cierto intervalo de minutos. De esta manera, el cerebro logra formar la asociación entre el contenido estudiado y el sonido.

Luego, cuando el usuario se vaya a dormir, se deberá colocar el cintillo en su cabeza. Este disposito monitoreará su actividad cerebral y enviará esta información a la app móvil. La app se encargará de identificar fases del sueño especificas y momentos precisos para poder reproducir el mismo sonido que fue asociado con el contenido estudiado.
